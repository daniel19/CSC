Other files

  • Script to mail files for homework submission mail-files. To use this secript, do the following setup:
    • Put a copy in your directory (probably simpler to use your browser's "save" rather than trying to cut and paste).
    • Use the command chmod u+x mail-files to tell the system that this is an executable script.
    You can now use it to mail me one or more files containing source code, for example: 
    	mail-files pgm1.scala pgm2.scala ....
    It prompts for homework number and text to include as the body of the message, and sends a Cc of the message to your username at trinity.edu. (If you don't get the Cc message, something probably went wrong. It may appear to come from a username that looks like a date, e.g., 2012-08-22. This is a result of our scheme for keeping track of when user accounts were created.)
