CSCI 3294 (Unix Power Tools):
Sample Programs

Shell scripts

  • upmachines: Script to display names of all machines that are "up".
  • lcname: Script to rename a file, changing all uppercase to lowercase.
  • rename-files: Script to do mass rename of files, changing suffix.
  • find-broken-links: Script to find broken symbolic links.
  • factorial: Script to compute n factorial.

vim configuration files

  • "Main" configuration file (goes in $HOME/.vimrc).
  • Configuration file for editing code (goes in $HOME/.vim/vimrc-for-code, start vim with vim -u $HOME/.vim/vimrc-for-code).
  • Common options, sourced by both of the above (goes in $HOME/.vim/vimrc-base).
